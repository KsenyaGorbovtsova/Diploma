# Diploma


<h1>IOS приложение FitShark.</h1>
<p>Создано для ведения личных тренировок.</p>
<p>Основные функции:<ul><li>Создание тренировок (+ удаление, редактирование)</li>
                 <li>Создание упражнений (+ удаление, редактирование)</li>
                 <li> Банк упражнений</li>
                 <li> Добавление пользователей в друзья</li>
                 <li> Отправка тренировок пользователям из списка друзей</li>
                 <li> Настройки профиля</li>
                  </ul>
                  </p>
